# Tracy

Tracy is an open-source Kotlin library that adds OpenTelemetry observability to LLM JVM applications. 
It provides APIs that help define what needs to be traced at a high level,
while hiding implementation details such as span structures and attribute names.
The library also supports multiple OpenTelemetry backends out of the box.
